# Gerenciador de Tarefas

Esta aplicação foi desenvolvida com React+Vite para avaliação à disciplina Laboratório de Desenvolvimento Web.


## Rodando o projeto

* Baixe o conteúdo do repositório
* Na raiz do projeto, rode > **npm install**
* Depois de instalar as dependências, rode **> npm run dev**
* Acesse **http://localhost:5173**
